# Jules Docs Site

This directory is a static documentation site for GitHub Pages and search indexing.

## Publish with GitHub Pages

1. Open the repository settings on GitHub.
2. Go to **Pages**.
3. Set the source to **Deploy from a branch**.
4. Choose `main` and `/docs`.
5. Save.

The expected project URL is:

```text
https://ckcreative.github.io/flutter_native_jules/
```

If the repository name changes, update these files:

- `docs/index.html`
- `docs/sitemap.xml`
- `docs/robots.txt`
- `README.md`

## Indexing Checklist

- Add the GitHub Pages URL as the repository website.
- Add a short repository description.
- Add topics such as `flutter`, `jules`, `developer-tools`, `ai`, `github`, and `code-review`.
- Submit the sitemap URL in Google Search Console after Pages is live.
- Link to the docs URL from launch posts, GitHub lists, newsletters, and community shares.
